# Tetris

ブラウザで遊べるシンプルなテトリスです。ビルドは不要です。

## 開き方

`index.html` をブラウザで開いてください。

スマホで遊ぶ場合は、PC側でこのフォルダを開いて次を実行します。

```bash
python3 -m http.server 8000 --bind 0.0.0.0
```

スマホを同じWi-Fiに接続し、ブラウザで `http://PCのIPアドレス:8000/` を開いてください。

## 操作

- `←` / `→`: 移動
- `↑`: 回転
- `↓`: ソフトドロップ
- `Space`: ハードドロップ
- `P`: 一時停止

スマホでは画面下のボタンで操作できます。
